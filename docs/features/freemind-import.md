# FreeMind import

Import `.mm` mind map files (FreeMind/Coggle format) into Obsidian Canvas.

## How to import

{% tabs %}
{% tab title="From file explorer" %}
Right-click a folder in the Obsidian file explorer and choose **Import mind map (.mm) to canvas**. Select the `.mm` file to import.
{% endtab %}

{% tab title="From command palette" %}
Open the command palette and run **Import mind map (.mm) file to canvas**. A file picker lets you browse for the `.mm` file.
{% endtab %}
{% endtabs %}

## What gets imported

* Node hierarchy (parent-child relationships)
* Node text content
* Edge connections with arrow indicators
* Tree structure and layout

The imported canvas is created in the same folder with the same name as the `.mm` file.
