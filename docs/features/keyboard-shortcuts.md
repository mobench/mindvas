# Keyboard shortcuts

Mindvas ships without default hotkeys. Configure them in **Settings** > **Hotkeys** by searching for "Mindvas".

## Available commands

| Command | Description |
| --- | --- |
| Edit selected node | Start editing the selected node's text |
| Add child node | Create a child node from the selected node |
| Add sibling node | Create a sibling node next to the selected node |
| Delete node and focus parent | Remove the selected node and select its parent |
| Flip branch to other side | Move a branch from right to left or vice versa |
| Toggle balanced layout | Distribute children evenly on both sides of root |
| Navigate right / left / up / down | Spatial navigation between nodes |
| Re-layout mind map | Recalculate layout for all trees |
| Layout forest | Arrange trees within the selected group into a grid |
| Resize & re-layout subtree | Resize nodes to fit content, then re-layout |
| Resize all nodes | Resize all nodes to fit their content |
| Detach subtree | Disconnect a branch into an independent tree |
| Toggle mindmap mode | Enable/disable mindmap mode for the current canvas |

## Suggested hotkey configuration

| Command | Suggested hotkey |
| --- | --- |
| Edit selected node | Enter |
| Add child node | Ctrl + . |
| Add sibling node | Ctrl + Enter |
| Delete node and focus parent | Ctrl + Shift + Backspace |
| Flip branch | Ctrl + Shift + S |
| Navigate right/left/up/down | Ctrl + Alt + Arrow keys |

{% hint style="info" %}
Mindvas includes a physical-key fallback for non-Latin keyboard layouts (Arabic, Hebrew, etc.). Shortcuts work regardless of your active input language.
{% endhint %}
