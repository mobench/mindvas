# Mouse actions

## Canvas interactions

### Nodes

| Action | Result |
|--------|--------|
| Drag a node | Move the node and all its descendants together |
| Alt+drag a node | Move only that node, leaving children in place |
| Alt+click a node | Select the entire tree (root + all descendants) |
| Ctrl+click a node | Zoom to fit the branch |
| Alt+click a connection point | [Insert a new node](../features/insert-between.md) between parent and child |
| Right-click a node | Context menu with "Copy node link" |

### Groups

| Action | Result |
|--------|--------|
| Alt+drag a group | Move the group without moving nodes from other trees |
| Right-click a group | Context menu with "Copy node link" and "Layout forest" |

### Edges

| Action | Result |
|--------|--------|
| Alt+click the purple connection dot | Insert a new node on that edge |

## Map outline panel

### Root nodes

| Action | Result |
|--------|--------|
| Click | Select and zoom to the node on canvas |
| Ctrl+click (ungrouped only) | Multi-select for grouping |
| Drag (grip handle) | Move tree to a different group or ungrouped area |
| Right-click (ungrouped, with selection) | "Create group" from selected roots |

### Groups

| Action | Result |
|--------|--------|
| Click header | Toggle collapse/expand |
| Double-click header | Rename group inline |
| Right-click header | "Rename group" and "Layout forest" |

### Toolbar

| Action | Result |
|--------|--------|
| Click search icon | Toggle search filter |
| Click collapse icon | Collapse or expand all groups |
