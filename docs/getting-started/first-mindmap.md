# Your first mind map

{% stepper %}
{% step %}
## Create a canvas

Right-click in the file explorer and choose **New canvas**, or use the command palette.
{% endstep %}

{% step %}
## Enable mindmap mode

Click the brain icon in the canvas toolbar to toggle mindmap mode on. The Map outline panel appears in the right sidebar.
{% endstep %}

{% step %}
## Create a root node

Double-click the canvas to create a text node. This is your root node.
{% endstep %}

{% step %}
## Add child nodes

Select the root node and press your configured "Add child" shortcut to create a child node connected by an edge. The layout engine positions it automatically.
{% endstep %}

{% step %}
## Keep building

Add siblings, children, and branches. The contour-based layout keeps everything tidy. Use the Map outline panel to navigate your tree.
{% endstep %}
{% endstepper %}
