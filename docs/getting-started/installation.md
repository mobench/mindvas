# Installation

{% stepper %}
{% step %}
## Open community plugins

Go to **Settings** > **Community plugins** > **Browse**.
{% endstep %}

{% step %}
## Search for Mindvas

Type "Mindvas" in the search bar and click **Install**.
{% endstep %}

{% step %}
## Enable the plugin

Toggle Mindvas on in the **Installed plugins** list.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
Mindvas requires Obsidian v1.0.0 or later.
{% endhint %}
